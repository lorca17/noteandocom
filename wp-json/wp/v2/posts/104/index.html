El modulo\u00a0requests de python se utiliza para conectarnos a diferentes paginas por diversos motivos.<\/p>\n
Este ejemplo se basa a la hora de hacer alg\u00fan bot o script el cual queremos que pase lo mas desbastecido posible.<\/p>\n
Para hacer esto tenemos que crear una cabecera convincente y no esta de mas a\u00f1adir un referer a nuestra cabecera.<\/p>\n
import requests\r\n\r\nuser_agent = 'Mozilla\/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit\/537.36 (KHTML, like Gecko) Chrome\/70.0.3538.77 Safari\/537.36'\r\n\r\nreferer = 'http:\/\/localhost\/vps640544\/noteandocom\/'\r\n\r\nheader = {'User-Agent': user_agent , 'referer': referer }\r\n \r\nhtml = requests.get('http:\/\/localhost\/vps640544\/noteandocom\/test-referer.php', headers = header, verify=False, stream=True)\r\nhtml.encoding = 'utf-8'\r\nprint (html.text) # Aqu\u00ed veras el c\u00f3digo fuente de la pagina\r\n<\/pre>\n
Informaci\u00f3n A\u00f1adida:<\/h2>\n
El User-Agent: Esta es una informaci\u00f3n que les tramite nuestro navegador a pagina que estamos visitando para mejorar nuestra experiencia como usuario entre otras funciones.<\/p>\n
El referer: Le informa a la pagina de donde llegamos, es decir la pagina donde hemos hecho click para llegar aqu\u00ed.<\/p>\n
