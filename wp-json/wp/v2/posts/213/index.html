Para cambiar los permisos del grupo necesitamos conectarnos por ssh, en windows puedes utilizar el programa putty.<\/p>\n

Aqu\u00ed unos ejemplos de como cambiar de forma r\u00e1pida los due\u00f1os de las carpetas, para no ir archivo por archovo se le a\u00f1ade \u00ab-R\u00bb si solo quieres cambiara un archivo no lo pongas.<\/p>\n

 <\/p>\n

sudo chown <usuario> -R <carpeta> (carpeta y todo su contenido)<\/p>\n

ejemplo: sudo chown web13 -R \/var\/www\/clients\/client2\/web13\/web\/plugins<\/p>\n

sudo chgrp <grupo> -R <carpeta> (carpeta y todo su contenido)<\/p>\n

ejemplo: sudo chgrp client2 -R \/var\/www\/clients\/client2\/web13\/web\/plugins<\/p>\n

Para hacerlo cambiar el grupo y el usuario de golpe se utiliza este comando:<\/p>\n

sudo chown usuario:grupo -R <carpeta> (carpeta y todo su contenido)<\/p>\n

ejemplo: sudo chown web13:client2 -R \/var\/www\/clients\/client2\/web13\/web<\/p>\n","protected":false},"excerpt":{"rendered":"
